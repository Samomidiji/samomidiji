Jacket

P5.js Development

This program suggests the right costume to wear depending on the temperature of the location. It was developed using the P5.js Library and uses device geolocation to determine the name, location and weather properties of a particular area.

Prity Trashy

P5.js Development

An interactive program developed using the P5.js library that allows users to save the planet by preventing trash from hitting the floor. Playable on both desktop and mobile.

Voice Race

P5.js Development

Race just with your voice. This program uses audio input and maps increase in volume to the velocity of the car. Developed using P5.js Library.

Alien Wars

P5.js Development

Spaceship and aliens. Battle to save earth by shooting the aliens. Uses both gyroscope and direction keys for movement. Touch screen shoots on mobile, spacebar on desktop.
